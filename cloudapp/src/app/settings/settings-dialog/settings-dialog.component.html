info

Here you can create an external search template. The format is: [Start of link][Search Criteria Type][End of link(optional)]


Example:

Original link: http://test.netpunkt.dk/search/work?search_block_form=term.default+adj+%22hans%20og%20grete%22&page_id=danbib-search

1. Start of link: http://test.netpunkt.dk/search/work?search_block_form=term.default+adj+%22

2. Search criteria Type (the part of the original link, you want Alma to provide): Title

3.End of link: %22&page_id=danbib-search
